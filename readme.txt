【系统需求】
=   运行平台WIN9X+PWS OR WINNT OR WIN2000 
=   数据库 ACCESS2000
=   编程脚本 ASP3.0
=   推荐：Windows 2000+IIS5.0+Access2000+IE5.5环境

【安装说明】
=   1、解压缩本系统到指定目录
=   2、更改数据库名称，并修改conn.asp中的数据库名称
=   3、将目录上传至你的空间
=   4、登陆地址http://…/AdRotator/
=   5、默认用户名及密码为admin和admin888
=   6、登陆后，请进入[系统设置]修改系统参数，如：用户名、密码等

【升级说明】
=   1、解压缩本系统到指定目录
=   2、备份你的数据库，将目录上传至你的空间
=   3、更改数据库名称，并修改conn.asp中的数据库名称
=   4、升级文件地址http://…/AdRotator/update.asp
